
# CrazePy

Python-like programming language, where C and Python syntax meet together.

CrazePy is a dynamically typed programming language, which means that a variable can have any data type, that doesn't need to be specified.

## Features
- [x] Variables
- [x] Functions
    - [x] Parameters
    - [x] callable
    - [x] returnable
- [x] Data types
    - [x] string
    - [x] int
    - [x] double
    - [x] float
    - [x] bool
- [x] Arithmetic operations
- [x] If statements
    - [x] Else statements
    - [x] Elif statements
- [ ] Standard Library
    - [x] Print function
    - [ ] Input function
    - [ ] ...
- [ ] For/While Loops
    - [x] while loops
    - [ ] for loops
- [ ] ...
